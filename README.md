# EEG Denoising Methods & Journals Survey

整理EEG去噪领域的传统方法、深度学习方法以及可投稿期刊调研结果。

## 项目结构

```
EEG_Denoising_Review/
├── README.md
├── data/
│   ├── EEG_denoising_traditional.csv   # 传统去噪方法
│   ├── EEG_denoising_deeplearning.csv  # 深度学习去噪方法
│   └── journals.csv                     # 可投稿期刊列表
```
